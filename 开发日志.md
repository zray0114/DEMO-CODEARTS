## 一、项目概述

### 1.1 系统功能
基于Vue3和Spring Boot构建的全栈物业费用管理系统，包含以下核心功能：

1. **用户管理** - 用户增删改查、登录认证
2. **费用管理** - 水费、电费、天然气费、物业费的缴费记录管理
3. **租户管理** - 租户信息管理、入住状态跟踪
4. **费用分摊** - 支持按租户数量和按入住天数两种分摊算法
5. **菜单管理** - 系统菜单配置
6. **数据统计** - 可视化图表展示
7. **图片上传** - 支持多张图片上传（二进制存储）
8. **Excel导出** - 费用分摊数据导出

### 1.2 技术架构

**前端技术栈：**
- Vue 3 (Composition API)
- Element Plus (UI组件库)
- Vue Router (路由管理)
- Axios (HTTP请求)
- XLSX (Excel导出)
- Vite (构建工具)

**后端技术栈：**
- Spring Boot 3.2.0
- MyBatis Plus 3.5.5 (ORM框架)
- MySQL 8.0 (数据库)
- Lombok (代码简化)

## 二、数据库设计

### 2.1 核心表结构

#### 用户表 (users)
```sql
CREATE TABLE users (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL,
    account VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

#### 费用表 (water_fees, electricity_fees, gas_fees, property_fees)
```sql
CREATE TABLE water_fees (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    payment_date DATE NOT NULL,
    amount DECIMAL(10,2) NOT NULL,
    images LONGBLOB,
    remark TEXT,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

#### 租户表 (tenants)
```sql
CREATE TABLE tenants (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    user_id BIGINT NOT NULL,
    check_in_date DATE NOT NULL,
    rent_amount DECIMAL(10,2) NOT NULL,
    deposit_amount DECIMAL(10,2),
    status TINYINT DEFAULT 1,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

#### 费用分摊表 (fee_shares)
```sql
CREATE TABLE fee_shares (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    tenant_id BIGINT NOT NULL,
    share_month VARCHAR(7) NOT NULL,
    water_fee DECIMAL(10,2),
    electricity_fee DECIMAL(10,2),
    gas_fee DECIMAL(10,2),
    property_fee DECIMAL(10,2),
    total_fee DECIMAL(10,2),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

```java
int tenantCount = tenants.size();
```

#### 算法二：按入住天数比例分摊
```java
long days = ChronoUnit.DAYS.between(effectiveStart, endDate) + 1;
BigDecimal ratio = BigDecimal.valueOf(days)
    .divide(BigDecimal.valueOf(totalDays), 6, RoundingMode.HALF_UP);
share.setWaterFee(totalWater.multiply(ratio).setScale(2, RoundingMode.HALF_UP));
```

### 3.4 图片上传功能

**前端实现：**


**后端处理：**
```java
// 多图合并为byte[]
if (images != null && images.length > 0) {
    int totalSize = 0;
    for (MultipartFile file : images) {
        totalSize += (int) file.getSize();
    }
    
    byte[] allImages = new byte[totalSize];
    int offset = 0;
    for (MultipartFile file : images) {
        byte[] bytes = file.getBytes();
        System.arraycopy(bytes, 0, allImages, offset, bytes.length);
        offset += bytes.length;
    }
    
    fee.setImages(allImages);
}
```

## 五、问题修复记录

### 5.1 编译错误修复

**问题：** Controller中实体类引用错误
**修复：** 
- ElectricityFeeController: WaterFee → ElectricityFee
- GasFeeController: WaterFee → GasFee
- PropertyFeeController: WaterFee → PropertyFee

### 5.2 图片上传错误修复

**问题：** Content-Type 'multipart/form-data' is not supported
**原因：** Controller使用@RequestBody接收JSON
**修复：** 改为@RequestParam + MultipartFile[]

### 5.3 费用分摊租户姓名显示修复

**问题：** 租户姓名字段为空
**原因：** 后端只返回tenantId，未关联查询
**修复：**
```java
Tenant tenant = tenantMapper.selectById(share.getTenantId());
if (tenant != null) {
    User user = userMapper.selectById(tenant.getUserId());
    if (user != null) {
        record.put("tenantName", user.getUsername());
    }
}
```

### 5.4 Excel导出错误修复

**问题：** blob URL无法访问
**原因：** Blob被提前释放
**修复：** 使用`<a download>`触发下载，延迟释放Blob URL

### 5.5 编辑图片回显错误修复

**问题：** newVal.map is not a function
**原因：** 编辑时images是byte[]而不是数组
**修复：** 添加Array.isArray检查，编辑时清空images
